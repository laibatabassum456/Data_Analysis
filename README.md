 📊 Data Science Mini Projects Portfolio

 📌 Overview

This repository contains a collection of data science tasks designed to build and demonstrate core skills in data analysis, visualization, and machine learning using Python. The projects cover both classification and regression problems and utilize popular libraries such as pandas, matplotlib, seaborn, and scikit-learn.

Each task follows a structured workflow including data loading, preprocessing, exploratory data analysis (EDA), model building, evaluation, and interpretation of results.

🌸 Task 1: Exploring and Visualizing the Iris Dataset

This task focuses on understanding and visualizing a simple dataset. The Iris dataset is used to explore relationships between different flower features such as sepal length, sepal width, petal length, and petal width.

Various visualizations such as scatter plots, histograms, and box plots are created to analyze patterns, distributions, and potential outliers. This task helps build foundational skills in data exploration and visualization.

 💳 Task 2: Credit Risk Prediction

This task involves predicting whether a loan applicant is likely to default or not. The dataset includes features such as applicant income, education, loan amount, and other financial attributes.

The process includes handling missing data, performing exploratory data analysis, encoding categorical variables, and training a classification model. The model’s performance is evaluated using accuracy and a confusion matrix.


 🏦 Task 3: Customer Churn Prediction

In this task, the goal is to identify bank customers who are likely to leave (churn). The dataset contains customer information such as age, balance, geography, and account details.

After cleaning and encoding the data, a machine learning classification model is trained. Feature importance analysis is also performed to understand which factors contribute most to customer churn.

 💰 Task 4: Insurance Claim Amount Prediction

This task focuses on predicting medical insurance charges based on personal attributes such as age, BMI, smoking status, and region.

A regression model is used to estimate insurance costs. The analysis also includes visualizing relationships between features and the target variable. Model performance is evaluated using metrics like Mean Absolute Error (MAE) and Root Mean Squared Error (RMSE).

📈 Task 5: Personal Loan Acceptance Prediction

This task aims to predict whether a customer will accept a personal loan offer. The dataset includes demographic and financial information such as age, job, marital status, and education.

The workflow includes data exploration, preprocessing, model training, and evaluation. Additionally, insights are extracted to identify which customer segments are more likely to accept loan offers.
